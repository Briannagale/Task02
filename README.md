# Train Reservation System – Java

This project is a simple Train Reservation System developed using Java. The system allows users to perform basic operations such as booking a train ticket, viewing reservation details, and cancelling reservations. The program works through a console-based interface where users can enter their details and choose different options from the menu.
The main objective of this project is to demonstrate the use of Java programming concepts such as classes, objects, conditional statements, loops, and user input handling. It helps in understanding how a basic reservation system works and how data can be managed in a simple application.
This project was developed as part of my Java Development internship task to improve programming logic and practical implementation of Java concepts.
